# UX e acessibilidade

Este projeto apresenta uma landing page simples e acessível sobre o tema "UX e acessibilidade: desenvolvendo interfaces inclusivas".

## Como visualizar

1. Abra o arquivo index.html em um navegador, ou
2. Inicie um servidor local na pasta do projeto com o comando:

```bash
python3 -m http.server 8000
```

Em seguida, acesse http://127.0.0.1:8000/.
